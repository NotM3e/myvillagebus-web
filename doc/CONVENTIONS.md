# Konwencje projektu

## Git Commits

- Krótki opis po angielsku, bez prefixów (feat:/fix:)
- Poziom B1+
- Przykład: "Add user management to admin panel"

## Komentarze w kodzie

- Język angielski (B1+)
- Tylko gdy nieoczywiste

## Komponenty

- Funkcyjne (hooks)
- MUI Icons dla ikon
- Tailwind + custom CSS variables (Material Design tokens)

## Nazewnictwo

- camelCase dla zmiennych/funkcji
- PascalCase dla komponentów
- snake_case dla kolumn w bazie (Supabase convention)
